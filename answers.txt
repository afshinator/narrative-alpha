### Layer 1: Ingestion & Extraction (The Scraper Intake)
* **Execution Boundary:** Communicates solely with the Bright Data proxy network.
* **Core Logic:** Accepts a target criteria string, runs automated web searches via Bright Data SERP API, retrieves full article bodies through Web Unlocker proxies, strips out boilerplate scripts, and outputs a standardized data manifest.
* **Decoupling Rule:** This layer contains no LLM code, graph concepts, or UI parameters. If Bright Data alters its JSON responses, changes are isolated entirely to this module.

### Layer 2: Cognitive Processing (The Knowledge Storage Plane)
* **Execution Boundary:** Interfaces with the Cognee Python SDK, DeepSeek API, and the attached storage volume.
* **Core Logic:** Ingests raw document arrays. Implements the JIT Split-Testing Entity Resolution strategy by split-routing raw source code to an unseeded workspace partition and a canonical normalized reference dictionary partition using DeepSeek-Chat.
* **Decoupling Rule:** This layer does not evaluate news bias or format dashboards. Foundational providers are controlled entirely via environment configuration flags.

### Layer 3: Analytical Deconstruction (The Forensic Metric Brain)
* **Execution Boundary:** Interfaces with premium LLMs (Claude 3.5 Sonnet).
* **Core Logic:** Consumes the structured graph JSON arrays generated in Layer 2. Executes comparative analysis prompts to determine intentional factual omissions, track linguistic framing transformations, calculate industry-stratified drift metrics, and evaluate Outlier Origin Provenance.
* **Decoupling Rule:** This module behaves as a stateless mathematical function. It reads data arrays and returns a single, comprehensive, frozen JSON intelligence payload.

### Layer 4: Presentation & Delivery (The User Interface Adapters)
* **Execution Boundary:** Read-only data consumption from Layer 3.
* **Core Logic:** Maps the structural payload directly to a web frontend client featuring independent URL pathways for each distinct event cluster.
* **Decoupling Rule:** This layer contains zero operational code, scraping logic, or data analysis steps. Modifying web styling will never break the data processing pipelines. (Note: Local Obsidian vault automated generation is scheduled as a Post-MVP feature enhancement).
